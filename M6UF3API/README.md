## Projecte desplegat a Vercel

https://m6uf3api.vercel.app/

---



---

### Llistar totes les entrades
GET  
https://m6uf3api.vercel.app/list

---

### Filtrar entrades per rang de dates
GET  
https://m6uf3api.vercel.app/filterdates/:dataini/:datafi

Exemple:  
https://m6uf3api.vercel.app/filterdates/2026-02-01/2026-02-28

---

### Afegir una nova entrada
POST  
https://m6uf3api.vercel.app/add

---

### Actualitzar una entrada per ID
PUT  
https://m6uf3api.vercel.app/update/:id

Exemple:  
https://m6uf3api.vercel.app/update/ID_ENTRADA

---

### Eliminar una entrada per ID
DELETE  
https://m6uf3api.vercel.app/delete/:id

Exemple:  
https://m6uf3api.vercel.app/delete/ID_ENTRADA
